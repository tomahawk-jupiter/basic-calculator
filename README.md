### Useful things I learnt

Stop text highlighting when clicking on text CSS:

    user-select: none;

Mouse finger pointer CSS:

    cursor: pointer;

make calculations within CSS:

    calc(10px + 20px);

Convert number string to number with unary operator plus:

    let num = '10';
    typeof(num);    // string
    typeof(+num);   // number
